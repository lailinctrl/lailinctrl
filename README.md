### About

I'm a full-stack engineer at **VCGamers**, where I help build a gaming marketplace and the loyalty + gamification surfaces wrapped around it. I started there as a Discord-bot-wielding marketing engineer, moved into React Native, and grew into full-stack — so I've seen the platform from a lot of angles.

I care about the unglamorous stuff that makes products feel solid: clean state machines, type-safe boundaries, motion that supports content instead of fighting it, and writing code that the next person on call can actually read.

### What I'm building

- 🎮 **[Gamification System](https://play.vcgamers.com)** — quests, rewards, progression, and engagement loops for the marketplace
- 🛒 **[VCGamers Marketplace](https://vcgamers.com)** — frontend features for a digital-products commerce platform
- 🪪 **[Seller App](https://seller.vcgamers.com)** — the merchant-facing surface for products, orders, and listings

The longer story (including archived projects, career arc, and the easter egg) lives on the [portfolio](https://portfolio-tau-gilt-69.vercel.app).
